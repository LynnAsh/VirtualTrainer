### Idea
I am trying to build a my own version of a tamagotchi pet game. I want to make this
because I really like playing games and grew up playing games inspired by
tamagotchis on my Nintendo DS.

This pet game will be used to just relax and play with once in a while.

### Effort
On the surface I think it seems simple enough but I know that I will get ideas to expand the project while working
on it that will take effort to create. Right now I already plan on making different minigames 
and chores for the user to do with their pet.
